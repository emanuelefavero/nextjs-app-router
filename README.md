# Next.js App Router

This is a cheat sheet project for the Next.js 13 [App Router](https://nextjs.org/docs/app)

> Note: Check out this repo's project to see the new features in action

## Getting Started

1. Clone the repo and `cd` into it
2. Run `npm i`
3. Run `npm run dev`
